# Print bsplines

Print bsplines

## Usage

``` r
# S3 method for class 'cpr_bs'
print(x, n = 6L, ...)
```

## Arguments

- x:

  a `cpr_bs` object.

- n, :

  number of rows of the B-spline basis matrix to display, defaults to
  6L.

- ...:

  not currently used.

## Value

the object `x` is returned invisibly
