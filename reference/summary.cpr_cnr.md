# Summarize Control Net Reduction Objects

Summarize Control Net Reduction Objects

## Usage

``` r
# S3 method for class 'cpr_cnr'
summary(object, ...)
```

## Arguments

- object:

  a `cpr_cnr` object

- ...:

  pass through

## Value

a `cpr_summary_cpr_cnr` object, that is just a `data.frame`
